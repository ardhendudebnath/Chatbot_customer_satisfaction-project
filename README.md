# Chatbot_customer_satisfaction-project

This project is focused on creating a chatbot which uses NLP techniques for analyzing the customer satisfaction. It is as a Chatbot that can process customer feedback and sentiment, to tell us about what level of satisfaction customer might have from our product.

Code Overview

This code is implemented in Python and uses a selection of libraries such as NLTK, NumPy, Pandas, Matplotlib and Scikit-learn. Project components

Clearly defined data loading: It begins by creating a dataset from a csv file, scale the text to all lower case and then it averages out from sentences to words.

Simple Greeting Response: Implements a basic greeting response, that, when tripped up by catching common greetings like "hello" or "hi", will reply.

Word Frequency Analysis-Applying code to check which word is repeating how many time in dataset and plot the top
